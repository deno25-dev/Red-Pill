# Red Pill Charting Database

Place your historical data folders and CSV files here. 
Connect this folder using the "Symbol Search" feature in the application to index all files for quick access.

Recommended Structure:
/database
  /BTCUSD
    BTCUSD_1h.csv
    BTCUSD_4h.csv
  /ETHUSD
    ETHUSD_1h.csv
